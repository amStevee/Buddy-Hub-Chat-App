# Buddy-Hub

## Project Overview

Buddy-Hub-Chat-App is an application that lets family, friend, and team members send text messages in a **fast** and **reliable** manner. It uses simple text, emojies, and symbols.

## Teach Stack

1. Frontend:
    * React
    * Tailwindcss
    * Redux tool kit
2. Backend:
    * Node.js
    * Express
3. Database: 
    * PostgreSQL
4. Realtime:
    * Socket.io
